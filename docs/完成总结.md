## 🚀 接下来要做什么？

### 方式一：一键启动（推荐新手）

**双击运行** `start.bat` 文件

它会自动：
1. 检查 Python 环境
2. 安装必要的依赖包
3. 启动 Jupyter Notebook

### 方式二：手动启动

#### 步骤 1: 安装依赖
打开终端，在项目目录下运行：
```bash
pip install -r requirements.txt
```

#### 步骤 2: 生成示例数据（可选）
```bash
python src/generate_sample_data.py
```

#### 步骤 3: 启动 Jupyter
```bash
jupyter notebook notebooks
```

#### 步骤 4: 开始分析
浏览器会自动打开，然后：
1. 先打开 `01_data_exploration.ipynb` 
2. 按顺序执行所有单元格
3. 完成后打开 `02_model_training.ipynb` 训练模型

## 🐛 常见问题

### Q1: 安装依赖失败？
```bash
# 升级 pip
python -m pip install --upgrade pip
# 重新安装
pip install -r requirements.txt
```

### Q2: Jupyter 无法启动？
```bash
# 重新安装 Jupyter
pip install --upgrade jupyter notebook
```

### Q3: 中文显示问题？
Notebook 中已设置中文字体，如仍有问题请确保系统安装了中文字体。

### Q4: 如何使用自己的数据？
1. 将数据文件放入 `data/raw/` 目录
2. 修改 Notebook 中的文件名
3. 调整列名匹配您的数据
