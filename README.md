# 大数据分析 - 应用负载规律研究

## 项目简介
本项目旨在通过大数据分析技术，研究和预测应用负载的变化规律。

## 项目结构
```
d:\bigdata\
├── data\              # 原始数据和清洗后数据
│   ├── raw\          # 原始数据
│   └── processed\    # 清洗后的数据
├── notebooks\         # Jupyter Notebook 分析文件
├── src\               # Python 源代码
│   ├── preprocessing\ # 数据预处理模块
│   ├── models\        # 机器学习模型
│   └── visualization\ # 可视化模块
├── configs\           # 配置文件
└── docs\              # 文档和报告
```

## 环境要求
- Python 3.8+
- Jupyter Notebook
- 相关数据分析库（pandas, numpy, scikit-learn 等）

## 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 查看 notebooks 中的示例分析
3. 运行预处理和模型训练脚本
